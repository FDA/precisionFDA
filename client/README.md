# PFDA Client App
This is the main webapp for the PFDA project.


Just some of the libraries used.
- typescript
- react
- styled-components
- webpack
- storybook
- react-query
- react-hook-form


## DEV Installation
1. `yarn install`
2. `yarn server`
3. open `https://localhost:4000`


## Storybook
1. `yarn storybook`
